# Milestone 2 Mock Assessment – Full Stack Project

##  Overview
This project demonstrates a full stack application using React (Frontend) and Node.js + Express (Backend) implementing Q1–Q10 requirements.

---

##  Frontend Features (React)
| Question | Feature |
|---------|---------|
| Q1 | ProductCard using Props |
| Q2 | LoginForm using useState + useRef |
| Q3 | UserStatus using useEffect with delay |
| Q4 | Routing + Dynamic User Details |
| Q5 | Window Resize detector using Event Listener |
| Q6 | Formik Login + Yup Validation |

Run frontend:
- cd frontend
- npm install
- npm start


## Backend Features (Node + Express)
| Question | Feature |
|---------|---------|
| Q7 | Basic API |
| Q8 | Logger Middleware |
| Q9 | Validation with express-validator |
| Q10 | JWT Authentication + Protected Route |

Run backend:
- cd backend
- npm install
- node index.js
