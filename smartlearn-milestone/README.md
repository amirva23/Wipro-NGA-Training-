## Prerequisites
- Node 18+, npm
- MongoDB running at mongodb://localhost:27017

## Backend
1. cd backend
2. copy .env (example provided)
3. npm install
4. node seed/seedCourses.js
5. npm run dev
6. Endpoints:
   - POST /api/courses
   - GET /api/courses
   - POST /api/enroll

## Frontend
1. cd frontend
2. npm install
3. npm start

## Tests
From backend:  npm test
