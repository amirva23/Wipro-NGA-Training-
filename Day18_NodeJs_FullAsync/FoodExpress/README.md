# Day 18 – Asynchronous Programming in Node.js

This project demonstrates three async techniques in Node.js using the FoodExpress Order System example.

# Topics Covered
- Callbacks  
- Promises  
- Async/Await  
- Callback Hell  
- Differences between Resolved and Fulfilled Promises  

# Files
- `callbacks.js` – Fetch order using callbacks  
- `promises.js` – Process payment using Promises  
- `asyncAwait.js` – Generate invoice using Async/Await  
- `differences.txt` – Theory explanation  

# To Run
-node callbacks.js
-node promises.js
-node asyncAwait.js
