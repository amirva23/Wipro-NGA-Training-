Student Skills Management API
SKILLTRACK 

A simple Express.js backend for managing students, their skills, and courses.
This project implements basic CRUD operations with routing and middleware.

# Features

- Fetch all students
- Get student by ID

# CRUD

- Create a new student
- Update existing student
- Delete a student

# Middleware for logging requests

-Middleware for validating request body and student ID
-In-memory storage (no database)
